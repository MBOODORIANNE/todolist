# ToDoList API (Spring Boot)

## Description
This is a **Spring Boot application** for managing a ToDo List.  
The project uses **H2 in-memory database** for storage and provides a **RESTful API** for performing CRUD operations on tasks.

---

## Features
- **CRUD operations** for tasks:
    - Create a new task
    - Read all tasks or a single task by ID
    - Update a task
    - Delete a task
- Mark tasks as **completed**
- Fully documented code with **Javadoc**
- H2 console available for database inspection

---

## Requirements
- Java 17 or higher (tested with JDK 21)
- Maven 3.9 or higher
